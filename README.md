# Tarea-Javascript-avanzado

Hola, ¿cómo están? Soy Tomás Bermúdez. Curso de Desarrollo en React JS. 181751

En este proyecto desarrollé un gestor de tareas. A través de él es posible cargar, agregar, buscar y filtrar tareas, poniendo en práctica el uso de clases, promesas y métodos de arrays.
Implemente mas mecánicas en JavaScript y mas estilos en el CSS, con el objetivo de seguir mejorando y salir de mi zona de comfort al no usar siempre los mismos estilos y códigos.

Instrucciones para clonar el repositorio y abrir la página

1 En GitHub, navegue hasta la página principal del repositorio.

2 Encima de la lista de archivos, haz clic en Code.

<img width="388" height="107" alt="image" src="https://github.com/user-attachments/assets/6359e5ac-050f-41ea-8748-f6425365deb6" />


3 Copia la dirección URL del repositorio.

<img width="453" height="337" alt="image" src="https://github.com/user-attachments/assets/e8db06cd-c8a1-496d-8963-39e9432fc265" />

4 Abra Git Bash.

5 Cambia el directorio de trabajo actual a la ubicación en donde quieres clonar el directorio.

6 Escriba git clone y pegue la dirección URL que ha copiado antes.

"git clone https://github.com/ttomas4/Tarea-Javascript-avanzado.git"

7 Presione Enter para crear el clon local.
